## 🏁 Getting Started

1. Clone repo
2. Run:

```bash
# First time install and start script
npm i && npm start
```

## ⚙️ Scripts

```bash
$ npm install

# Start server on http://localhost:3000
$ npm start

$ npm build

$ npm test

$ npm eject
```

## ✔️ Deployment

Deployed on netlify from master branch
